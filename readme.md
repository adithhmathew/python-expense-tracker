# Expense Tracker (Python)

## Description
A simple command-line expense tracker built using Python.  
It allows users to add, view, and analyze their expenses.

## Features
- Add expenses (amount, category, description)
- View all expenses in clean format
- Calculate total spending
- Category-wise summary
- Data stored using CSV (persistent storage)

## Technologies Used
- Python
- CSV module

## How to Run
1. Open project in PyCharm or any IDE
2. Run `main.py`
3. Use menu to interact
